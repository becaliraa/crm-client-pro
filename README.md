# 💻 CMClient Pro

Sistema web desenvolvido para gerenciamento de clientes e automação de processos digitais.

O projeto foi criado com o objetivo de praticar desenvolvimento front-end moderno utilizando React, TypeScript e TailwindCSS, aplicando organização de componentes e boas práticas de desenvolvimento.

---

# 🚀 Tecnologias Utilizadas

Vite
React
TypeScript
TailwindCSS
Node

# 📂 Estrutura do Projeto

src  
 ├ components → componentes reutilizáveis da interface  
 ├ context → gerenciamento de estado global com Context API  
 ├ hooks → hooks personalizados  
 ├ lib → funções utilitárias e helpers  
 ├ pages → páginas da aplicação  
 ├ services → integração com APIs e lógica de serviços  
 ├ test → arquivos de teste  
 ├ types → tipagens TypeScript  
 ├ App.tsx  
 ├ main.tsx  
 ├ app.css  
 └ index.css  

Outros arquivos importantes:

public → arquivos públicos  
package.json → dependências do projeto  
package-lock.json → controle de versões das dependências  
vite.config.ts → configuração do Vite  
tailwind.config.ts → configuração do TailwindCSS

# ⚙️ Como Rodar o Projeto

Clone o repositório:

git clone https://github.com/becaliraa/crm-client-pro

Entre na pasta do projeto:

cd nome-do-projeto

Instale as dependências:

npm install

Execute o projeto:

npm run dev

O projeto ficará disponível no navegador local.
